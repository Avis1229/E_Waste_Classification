üìã PROJECT GOAL
---------------
Build a CNN model to classify e-waste into 6-8 categories for automated recycling

üóÇÔ∏è PHASE 1: DATA (Week 1)
--------------------------
‚ñ° Find dataset on Kaggle or create custom (min 500 images/class)
‚ñ° Organize: data/train, data/val, data/test (80/10/10 split)
‚ñ° Check for: corrupted images, class imbalance, data quality
‚ñ° Setup augmentations: albumentations library
‚ñ° Create dataset.py with PyTorch DataLoader

üìÅ Categories: batteries, cables, chargers, circuit_boards, keyboards, phones, monitors, misc

ü§ñ PHASE 2: MODEL (Week 2)
---------------------------
‚ñ° Baseline: Simple 3-layer CNN (establish minimum performance)
‚ñ° Transfer Learning: EfficientNetB0 or MobileNetV3
  - Freeze base layers
  - Replace head with custom classifier
  - Input: 224x224x3, Output: num_classes
‚ñ° Training setup:
  - Loss: CrossEntropyLoss
  - Optimizer: Adam (lr=1e-4)
  - Scheduler: ReduceLROnPlateau
  - Early stopping (patience=5)
  - Save best model: models/best_model.pth

üìä PHASE 3: TRAINING & EVAL (Week 2-3)
---------------------------------------
‚ñ° Train 30-50 epochs with GPU
‚ñ° Track: loss, accuracy, F1-score per epoch
‚ñ° Use TensorBoard or print logs
‚ñ° Evaluate on test set:
  - Confusion matrix
  - Per-class precision/recall/F1
  - Inference time (target: <100ms)
‚ñ° Error analysis: why did model fail?
‚ñ° Hyperparameter tuning if needed
